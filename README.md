# Scripts

`2emacs.py` Launch command line in emacs

`atomsay.py` Maintain podcast archives. Download mp3s from RSS. Can also convert text RSSs to speech if /usr/bin/say installed

`bathist.py` Calculate average battery discharge speed from upowerd data

`caps2altgr.desktop` Desktop autostart entry that remaps Caps Lock to AltGr via xmodmap

`crash-dumps-win7-setup.bat` Configure Windows Error Reporting to save crash dumps to user profile

`crashtools.py` Analyse user mode stack or full memory dumps

`cxxwrapper.py` Compiler wrapper. Adds map file output to link command line

`dmn.js` Join Windows machine to Active Directory domain with UAC elevation

`drtf.py` Parse RTF files and print structure tree with hex character decoding

`du_obj.py` Break down section size contributions in binaries or compare symbol/section sizes between binary versions

`emacstools.py` LLDB Python script for decoding Emacs Lisp objects and stack frames

`etags.sh` Run etags on all non-test source files in a repo and load tags into Emacs

`fmacro.tcl` Find C++ macro definitions by running g++ -E

`gforks.py` Query GitHub API to list every branch in every fork with unique commits for a project

`git-add-progress.tcl` Show live progress indicator while git add is running

`jsontab.js` Flatten JSON file into tabular format showing leaf values with full key paths, grep-friendly

`lslg.vbs` List all Windows local groups and their members via WinNT provider

`Makefile` Flags to build samba, thunar, glib etc.

`mousetrace.py` eBPF script to trace mouse click latency between kernel event and user-space GTK processing

`p2xfce.sh` Configure and launch XFCE session for GPD Pocket 2 with screen rotation, trackpoint, and USB keys

`path.js` Modify %PATH% in Windows registry with UAC elevation

`procx.d` DTrace script to print SIGSEGV handler address and RLIMIT_CORE per process

`rebase-outside.sh` Perform git rebase in a separate worktree to keep main working copy file mtimes untouched

`rundisk_vb.sh` Create VirtualBox VM from raw disk device using raw VMDK, with optional DVD attachment

`skvb.py` Send keyboard input to VirtualBox VM by mapping text to scancodes and key events

`smallrd.sh` Build minimal (23MB) initramfs with dracut for booting GPD Pocket 2

`sync.sh` Sync two directories bidirectionally via putting all content to git

`tee.js` Duplicate stdin to both stdout and a file

`term.pl` Launch xterm with randomized 16-color palette sorted by brightness

`tmpshare.sh` Quickly share a directory in a new smbd process

`trace.py` LLDB script for tracing function calls, printing stacks, and running custom commands at tracepoints

`unmkslf.py` Unpack MakeSelf self-extracting archives without executing embedded shell script

`usbkeys.py` Manipulate kernel scancode tables and trace USB keyboard events for GPD Pocket Japanese keyboard

`wdisk.sh` Boot Linux disk images or block devices in an average QEMU configuration with OVMF EFI or BIOS

`Workplace.yml` Ansible file to install bash aliases
